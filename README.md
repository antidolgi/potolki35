# Потолки35 — сайт премиальных натяжных потолков

Одностраничный лендинг, построенный на статическом стеке (HTML, CSS, JS) с эффектами.  
Живое демо: [https://username.github.io/potolki35]

## Особенности

- **Имитация натяжения полотна** на главном экране (GSAP + Lottie)
- **3D-карточки фактур** с эффектом параллакса (VanillaTilt)
- **Калькулятор стоимости** с анимацией перелистывания цифр
- **Слайдер сравнения «до/после»** (кастомный)
- **Кастомный курсор-лазер**
- **Форма обратной связи** с отправкой в Telegram
- Плавный скролл (Lenis) и анимации появления секций
